Multilingual Speech Synthesis with Data Augmentation

Abstract

State of the art text-to-speech (TTS) models can generate high fidelity monolingual speech, but it remains a challenging problem to generate multilingual speech from a given speaker identity. One major hurdle is that the training data used in TTS systems is usually from monolingual speakers because it is extremely hard to source speakers with multilingual capabilities at the native level. Subsequently, language and speaker infor- mation are highly correlated which makes it very challenging for the model to disentangle their conditioning features both during training and at inference time. In this paper, we propose a data augmentation approach to tackle this problem. We leverage the power of a pre-trained cross-lingual representation model to extract speaker-independent context features and use these features as input to a voice conversion model. The voice con- version model is used to convert multilingual and multi-speaker speech data to the voice of the target speaker. A single-speaker, multilingual TTS is trained using the target speakerâ€™s data from their native language together with the synthetic multilingual data converted from other speakers. Through augmenting data from four other languages, we built a multilingual TTS voice that speaks 5 languages(English, French, German, Italian and Spanish). Our system achieves improved Mean Opinion Score (MOS) compared with the baseline of multi-speaker system for all languages, specifically: 3.74 vs 3.62 for Spanish, 3.11 vs 2.71 for German, 3.47 vs 2.84 for Italian, and 2.72 vs 2.41 for French.
